Buttons

Source: stylemark/stylemark_markdown_buttons.md

Buttons can be used with <a>, <button>, and <input> elements. Buttons have different sizes:

Example 
<link href="https://fonts.googleapis.com/css?family=Roboto:300,400" rel="stylesheet">
<style>
  .btn { margin: 5px 10px;}
</style>

<button class="btn btn-primary btn-lg">Large</button>
<button class="btn btn-secondary btn-md ">Base With Borders</button>
<button class="btn btn-primary btn-md ">Base</button>
<button class="btn btn-primary btn-sm">Small</button>
<button class="btn btn-primary btn-xs">Extra Small</button>

Sizes

Source: stylemark/stylemark_markdown_sizes.md

To define size of an element we use bootstrap's grid system col-md-n/col-xs-n, where n is the amount of twelfth part of a width of the parent element (the whole bootstap styles are implemented, so feel free to use it)

Example 
<link href="https://fonts.googleapis.com/css?family=Roboto:300,400" rel="stylesheet">

<div class='row'>
  <div class='col-md-12'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
</div>
<div class='row'>
  <div class='col-xs-6'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
  <div class='col-xs-4'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
  <div class='col-xs-2'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
</div>
