Buttons

Source: stylemark/stylemark_markdown_buttons.md

Buttons are used to make actions. It should be noticeable and easy to perform with one click or tap. Users can use it to navigate, or take action.

Button options

* Primary button: Used as a main action on the interface. Provides extra visual weight and identifies the primary action in a set of buttons. * Secondary button: Used as when there are more than one action and as a secondary action on the interface or when the action are not main focus. * Info button: Contextual button for informational messages and drive users to learn more. * Warning button: Use to indicates caution should be taken with this action. The action may not be undo. * Danger button: Use to indicates a dangerous or potentially negative action. The action may not be undo.

Example 
<link href="https://fonts.googleapis.com/css?family=Roboto:300,400" rel="stylesheet">
<style>
  .btn { margin: 5px 10px;}
</style>

<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-secondary disabled">Disable</button>
<button class="btn btn-info">Info</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-danger">Danger</button>

Button tags

Buttons can be used with <a>, <button>, <div> and <input> elements.

Example 
<link href="https://fonts.googleapis.com/css?family=Roboto:300,400" rel="stylesheet">
<style>
  .btn { margin: 5px 10px;}
</style>

<a class="btn btn-primary btn-lg">Link</a>
<button class="btn btn-primary btn-lg ">Button</button>
<div class="btn btn-primary btn-lg ">Div</div>
<input type="submit" class="btn btn-primary btn-lg">Input</input>

Sizes

Source: stylemark/stylemark_markdown_sizes.md

To define size of an element we use bootstrap's grid system col-md-n/col-xs-n, where n is the amount of twelfth part of a width of the parent element (the whole bootstap styles are implemented, so feel free to use it)

Example 
<link href="https://fonts.googleapis.com/css?family=Roboto:300,400" rel="stylesheet">

<div class='row'>
  <div class='col-md-12'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
</div>
<div class='row'>
  <div class='col-xs-6'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
  <div class='col-xs-4'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
  <div class='col-xs-2'>
    <input class='form-control' placeholder="have you smiled today?"/>
  </div>
</div>
